# Skill: j7_optimizer

**Agent**: Content (Léo)
**Priority**: P1

## Purpose

Optimize the J7 (Day 7) experience to maximize the aha moment and reduce early churn.

## Why J7 matters

Day 7 is the critical survival threshold for SaaS:
- Users who reach the aha moment by J7 have 4× higher 90-day retention
- 60-80% of churn happens before J30 — most of it before J7
- If a user hasn't gotten value by day 7, they almost certainly won't convert

## Aha moment definition

For each SaaS, define the aha moment:
- What specific action = user has received clear value?
- How long does it take today? (time-to-value)
- What % of users complete it within 7 days? (baseline)

## Optimization framework

### Step 1: Identify the friction
- Map the activation funnel step by step
- Find the biggest drop-off point
- Common culprits: blank state, complex setup, missing data import

### Step 2: Reduce time-to-value
- Default data/templates so users see value immediately (no blank state)
- Guided onboarding (max 3 steps)
- Skip optional setup — get to value first, configure later

### Step 3: Email trigger sequence (J0-J7)
- J0: immediate "start here" email with ONE action
- J1: check if action completed → different path if not
- J3: "here's what you could do next" (feature 2)
- J7: personal check-in from Karim (personalized, 1:1 feel)

### Step 4: In-app nudges
- Empty state with sample data + "import yours in 30 seconds"
- Progress bar toward aha moment
- Contextual tooltips (not tutorials)

## Output

J7 activation audit document with:
- Current activation rate
- Identified friction points (ranked)
- 3 specific changes to implement this week
- Expected activation rate improvement
