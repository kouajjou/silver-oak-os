# Skill: landing_copy

**Agent**: Content (Léo)
**Priority**: P0

## Purpose

Write conversion-optimized landing page copy for a SaaS product.

## Page structure

### Above the fold
1. **Headline** — biggest benefit, max 8 words, verb-first ("Turn [pain] into [gain]")
2. **Subheadline** — expands the headline in 1-2 sentences with specificity
3. **CTA button** — action verb + benefit ("Start free trial" or "See how it works")
4. **Social proof** — 1 line ("Used by 200+ founders" or "⭐️⭐️⭐️⭐️⭐️ on Product Hunt")

### Problem section
- 3 specific pains the customer feels (not features, pains)
- Make them feel "yes, that's exactly me"

### Solution section
- 3 features mapped to the 3 pains (problem→solution pairs)
- Screenshot or GIF for each

### Social proof section
- 3 testimonials with: full name, role, company, photo
- 1 specific result in each quote ("saved 3 hours/week", "closed $50K deal")

### Pricing section
- 2-3 tiers, most popular highlighted
- Clear list of what's included/excluded
- Money-back guarantee if applicable

### FAQ section
- 5-7 questions addressing top objections (pricing, security, integrations)

### Final CTA
- Repeat the primary CTA with urgency if applicable

## Guard-rails

- Never use "AI-powered" without explaining what the AI actually does
- Never use "revolutionary", "game-changing", "disruptive"
- Always include a human name and photo on testimonials (not "Marketing Manager")
- Pricing must match Elena's Stripe configuration
