# Skill: demo_video

**Agent**: Content (Léo)
**Priority**: P1

## Purpose

Create a video demo brief that communicates the product's core value in under 2 minutes.

## Demo video structure

### 60-second "elevator demo" (primary)
- **0-5s**: The hook — state the pain in 1 sentence
- **5-15s**: Introduce the product (name + tagline)
- **15-50s**: Core feature demo — show the aha moment
- **50-60s**: CTA (sign up, trial, etc.)

### 90-second "feature demo" (secondary)
- Extended to show 2-3 additional features
- Useful for Product Hunt gallery

## Script format

```markdown
## DEMO SCRIPT: [Product Name]

**Format**: Screen recording + voiceover
**Duration**: 60 seconds
**Tool**: Loom / QuickTime / OBS

### Scene 1 (0-5s)
[VISUAL]: [what's on screen]
[VOICE]: "[Hook — the pain]"

### Scene 2 (5-15s)
[VISUAL]: [product logo / landing]
[VOICE]: "[Solution intro]"

### Scene 3 (15-50s)
[VISUAL]: [live product demo]
[VOICE]: "[Narrate the aha moment]"

### Scene 4 (50-60s)
[VISUAL]: [signup screen / pricing]
[VOICE]: "[CTA]"
```

## Production tips

- No talking head — screen recording only (faster to produce)
- Captions mandatory (85% of videos watched on mute)
- Music: royalty-free, neutral background
- Subtitles font: min 20px, high contrast
- Zoom in on key UI elements

## Distribution

Post to: Product Hunt gallery, landing page hero, LinkedIn video post, Twitter/X.
