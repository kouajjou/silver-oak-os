Mode UI/UX Pro Max - Review et amélioration complète de l'interface.

## Cible: $ARGUMENTS
(Page, composant, ou "full app")

## Analyse UX approfondie

### 1. User Journey Analysis
- Identifier les parcours utilisateur principaux
- Mapper chaque étape du parcours
- Identifier les points de friction
- Proposer des améliorations de flow

### 2. Information Architecture
- Hiérarchie de l'information claire ?
- Navigation intuitive ?
- Groupement logique des éléments
- Labels et textes clairs

### 3. Visual Design System
- Cohérence des couleurs (palette définie)
- Typography scale (tailles, weights, line-heights)
- Spacing system (4px/8px grid)
- Iconographie cohérente (Lucide Icons)
- Ombres et élévations
- Border radius consistency

### 4. Interaction Design
- Feedback visuel sur chaque action
- Transitions et animations (subtiles, purposeful)
- Loading states (skeleton, spinner, progress)
- Error states (inline, toast, modal)
- Success states (confirmation, redirect)
- Empty states (illustration, CTA)

### 5. Micro-interactions
- Hover effects
- Focus states
- Active states
- Disabled states
- Tooltips si nécessaire

### 6. Mobile Experience
- Touch-friendly (48px minimum touch targets)
- Thumb-zone optimization
- Bottom navigation (si applicable)
- Swipe gestures (si applicable)
- Pull-to-refresh (si applicable)

### 7. Copywriting
- Microcopy clair et actionable
- Error messages utiles (pas techniques)
- CTA buttons avec verbes d'action
- Placeholder text informatif

## Output
- Audit complet avec score /100
- Top 5 quick wins (effort minimal, impact maximal)
- Mockup textuel des améliorations
- Code TailwindCSS pour chaque amélioration
- Priorités: P0 (critique UX) / P1 (amélioration) / P2 (polish)
