🎯 Pour Maestro — comment utiliser Mode 3

Quand un test frontend est demandé, Maestro doit appeler le tool browser_test via le MCP Bridge Factory. Exemple :

Tool: browser_test
Arguments:
  url: "http://localhost:3010/agent/maestro"
  scenario: "Test page agent Maestro charge correctement"
  screenshot: true
  device: "desktop"
  assertions:
    - "Title contains Silver Oak"
    - "URL contains agent/maestro"
    - "Text exists Maestro"

Maestro reçoit en retour le résultat JSON structuré avec status PASS/FAIL/PARTIAL/ERROR + screenshot path.
