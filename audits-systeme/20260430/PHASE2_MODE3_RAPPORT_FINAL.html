🐛 Bug réel détecté — pour claude-frontend à fixer

1 bug a11y détecté sur http://localhost:3010/ :

[moderate] meta-viewport must allow zoom

Le frontend Next.js a un meta viewport qui empêche les utilisateurs de zoomer :

<meta name="viewport" content="width=device-width, initial-scale=1, maximum-scale=1, user-scalable=no">

Fix recommandé :

<meta name="viewport" content="width=device-width, initial-scale=1">

📍 Probablement dans silver-oak-os/frontend/app/layout.tsx ou app/(...)/layout.tsx
