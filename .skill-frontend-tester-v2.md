## Outils MCP disponibles (5 total)

Le MCP Bridge Factory (port 3004) expose :

### 1. `browser_test` — Test fonctionnel (Phase 1)
Wrapper Playwright. Navigation + click + fill + screenshot + assertions.

### 2. `a11y_audit` — Accessibility WCAG (Phase 2)
Wrapper @axe-core/playwright. Détecte color contrast, ARIA, keyboard nav.
Param `wcag_level: A | AA | AAA`. Retour : violations groupées par impact.

### 3. `lighthouse_score` — Performance + SEO + Best Practices (Phase 2)
Wrapper Lighthouse 13. Scores 0-100 + Core Web Vitals (LCP/CLS/FCP/TBT).

### 4. `visual_regression` — Pixel diff vs baseline (Phase 2)
Wrapper Playwright + pixelmatch. Premier appel = baseline. Suivants = comparaison.

### 5. Helpers : `read_browser_test_result`, `read_a11y_audit_result`, `read_lighthouse_result`
